# Учебный проект: Revo Coffee

Этот проект является каркасом для будущего веб-приложения кофейни "Revo". В рамках практического занятия 2 была выполнена семантическая HTML-разметка по предоставленному макету.

Сайт включает в себя:
- Шапку (`<header>`) с логотипом и навигацией.
- Основной контент (`<main>`), разделённый на секции.
- Подвал (`<footer>`) с копирайтом.

Проект создан для закрепления навыков работы с семантической разметкой, структурированием кода и публикацией на GitHub Pages.



## Технологии

HTML5 — для создания семантической структуры страницы.
Git — для контроля версий.
GitHub Pages — для публикации сайта.
